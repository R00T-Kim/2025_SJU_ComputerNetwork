# Project Contribution Log

This document details the individual contributions of each team member to the "2025_SJU_ComputerNetwork - Simple HTTP Chat Project". This log serves as evidence of contribution for the individual scoring component of Homework 3 (15 pts).

Each team member should clearly and concisely describe their specific roles, responsibilities, and tasks completed in the following areas:

-   **Design & Planning:** Contribution to architectural decisions, protocol choices, system design, data structures, and overall project planning.
-   **Backend Implementation (Python):** Specific features or modules developed for the Python HTTP server (e.g., socket handling, HTTP parsing, routing, channel management, file upload logic).
-   **Frontend Implementation (React.js):** Specific features or components developed for the React application (e.g., UI design, chat interface, message display, user input handling, API integration, file upload/download UI).
-   **Testing & Debugging:** Involvement in writing tests, identifying and resolving bugs, performance optimization.
-   **Documentation:** Creation or significant updates to `README.md`, `CONTRIBUTION.md`, code comments, or other project documentation.
-   **Deployment & Configuration:** Contribution to setting up the development environment, build scripts (`start.sh`, `start.bat`), or deployment procedures.
-   **Other:** Any other significant contributions not covered above.
